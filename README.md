# Mortgage vs Investment Calculator

Интерактивный калькулятор, позволяющий сравнить две стратегии управления деньгами при ипотеке:

1. **Досрочное погашение ипотеки**
2. **Инвестирование свободных средств**

Калькулятор учитывает:

- аннуитетный платеж по ипотеке
- досрочные погашения
- инвестиционную доходность
- инфляцию
- налоговые вычеты
- инвестиционный вычет ИИС
- имущественный вычет
- вычет по процентам
- двух супругов с разными доходами

Результатом расчета является **помесячная модель на весь срок ипотеки**.

---

# Возможности

Калькулятор рассчитывает:

- остаток долга по ипотеке
- проценты по кредиту
- тело кредита
- инвестиционный капитал
- инвестиционный доход
- налоговые возвраты
- момент закрытия ипотеки
- момент, когда инвестиции перекрывают долг

Также отображаются:

- реальные ставки с учетом инфляции
- суммарные проценты по ипотеке
- суммарные налоговые возвраты
